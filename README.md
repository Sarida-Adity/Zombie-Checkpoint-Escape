## Project Description

Zombie Checkpoint Escape is a 3D OpenGL-based game where the player moves through checkpoints, solves color-tile puzzles, avoids zombies, and clears walls to win the game.

## Features

- 3D OpenGL environment
- Player movement and rotation
- Zombie chasing system
- Color-based checkpoint puzzles
- Timer and life system
- First-person and third-person camera
- Restart option

## Controls

- W: Move forward
- S: Move backward
- A: Rotate left
- D: Rotate right
- 1/2/3/4: Select color tile
- Right Mouse Click: Toggle camera view
- Arrow Keys: Move camera
- R: Restart game
